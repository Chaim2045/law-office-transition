## ✨ תכונות עיקריות

### 🔄 Autosave חכם
- שמירה אוטומטית 1.5 שניות אחרי הפסקת הקלדה
- שמירה מיידית בעזיבת שדה (blur)
- מניעת שמירות מקבילות

### 💾 Dual Persistence
- שמירה ראשית ל-**Firebase Realtime Database**
- גיבוי אוטומטי ל-**localStorage**
- תמיכה מלאה ב-**offline mode**

### 🎨 אינדיקטורים ויזואליים
- **💾 Saving** - כתום, מופיע בזמן שמירה
- **✓ Saved** - ירוק, מופיע אחרי שמירה מוצלחת
- **❌ Error** - אדום + כפתור "נסה שוב"
